Shop by Look is a WooCommerce-dependent WordPress plugin that allows the user to bulk-buy grouped products and view product groups.

== Description ==

Purpose of plugin is to group WooCommerce products on one page and let your customers to look at picture that you have set for group of products and to buy all at once, or select only products that he need from that picture. After you selected products, they will appear automatically in shopping cart( from there your customers can chose desired amount and fill data related to billing and shipping -> This is functionality of Woocommerce ).

= Benefits =
Benefits are that you can advertise and sell more products at once. You can make attractive various combinations of your products for your customers. Example: You sell clothing on your Wordpress site, you can make combination with Jacket, Pants, and Shirt ( Make picture with those 3 products on some model ), and sell together those products.

= Third party API-s and Data protection =
This plugin is not connected to any third party API, and does not send data to any other service. So privacy of your data depends on your server security and Wordpress configuration.

== Installation ==

1. Download Woocommerce( https://wordpress.org/plugins/woocommerce/ )( desired version is 2.5.2 ) and upload to `/wp-content/plugins/` directory
2. Download Shop by Look plugin and upload to `/wp-content/plugins/` directory
3. Activate Woocommerce plugin through the 'Plugins' menu in WordPress
4. Activate Shop By Look plugin through the 'Plugins' menu in Wordpress

== Frequently asked questions ==

= I can not activate plugin. What can be problem? =

Please check is Woocommerce plugin exists, and is it activated in 'Plugins' Wordpress section. If don't exists or its not activated, please follow installation instructions.

= How to create new shop by look product page? =

Go to Shop by Look menu item from Wordpress menu in Dashboard. Click add new, next add title, content, featured image, next enter exact name of your Woocommerce product in meta field bellow, after that product is added to shop by look page.
